# Top-Courses

A React.js project that fetches and displays a list of top courses using an API. Users can view course details dynamically from the API.

# Features
 1. Fetch data from API and display courses dynamically. 
 2. Responsive design for mobile and desktop. 
 3. Simple and clean UI with cards for each course.

# Tech Stack
- **React.js**
- **Tailwind CSS (for styling) Fetch API (for data fetching)**
